# FiveM Custom HUD 🎮

Кастомный HUD для GTA V (FiveM) с динамической информацией о локации и красивыми волнами голоса.

## 📋 Возможности

- ✅ **Динамические улицы** - показывает текущую локацию игрока
- ✅ **Компас на русском** - С, В, Ю, З, СВ, ЮВ, ЮЗ, СЗ
- ✅ **Время в реальном времени** - из игровых часов
- ✅ **Температура** - отображение температуры
- ✅ **Волны голоса** - красивая анимация при разговоре
- ✅ **Полная кастомизация** - все позиции и цвета в config.lua

## 📁 Структура файлов

```
fivem-hud/
├── fxmanifest.lua    # Манифест ресурса
├── config.lua        # Конфигурация позиций и цветов
├── utils.lua         # Утилиты (зоны, направления)
├── client.lua        # Основной скрипт HUD
└── README.md         # Этот файл
```

## 🚀 Установка

1. **Скачай папку** или клонируй репозиторий:
```bash
git clone https://github.com/nekitdda/fivem-hud.git
```

2. **Переместить в resources:**
```
resources/
└── fivem-hud/
    ├── fxmanifest.lua
    ├── config.lua
    ├── utils.lua
    └── client.lua
```

3. **Добавь в server.cfg:**
```
ensure fivem-hud
```

4. **Перезагрузи сервер:**
```
refresh
start fivem-hud
```

## ⌨️ Команды

| Команда | Описание |
|---------|----------|
| `/hudon` | Включить HUD |
| `/hudoff` | Выключить HUD |
| `/voicetest` | Включить/выключить тест волн голоса |

## ⚙️ Кастомизация

Отредактируй `config.lua` для изменения:

### Позиции элементов
```lua
Config.HUD = {
    time = {
        x = 960,  -- X координата
        y = 50,   -- Y координата
        scale = 0.7,
        color = {255, 255, 255, 255}  -- R, G, B, A
    },
    -- и остальные элементы...
}
```

### Цвета
```lua
color = {255, 255, 255, 255}  -- RGB + Alpha (0-255)
```

### Направления
```lua
Config.Directions = {
    ['N'] = 'С',   -- Твой текст
    ['S'] = 'Ю',
    -- и т.д.
}
```

## 🎨 Элементы HUD

- **Время** - вверху по центру (8:24 PM)
- **PD статус** - зелёная точка вверху
- **Температура** - справа вверху (67°F)
- **Компас** - направление (С, В, Ю, З)
- **Улица** - текущая локация внизу слева
- **Волны голоса** - красивая анимация по центру внизу

## 🎯 Интеграция с Voice Chat

Для работы волн голоса с реальным voice script'ом раскомментируй в `client.lua`:

```lua
if GetResourceState('pma-voice') == 'started' then
    RegisterNetEvent('pma-voice:volumeUpdate', function(talking)
        VoiceActive = talking
    end)
end
```

Адаптируй под свой voice script.
